Judul Project
Website Pemesanan Kopi “Kopi Santai”

Deskripsi Singkat
Kopi Santai adalah sebuah website pemesanan kopi berbasis web yang dibuat sebagai mockup/prototype. Website ini memungkinkan pengguna untuk melihat menu kopi, mendapatkan rekomendasi kopi berdasarkan mood, melakukan custom kopi sesuai selera, menambahkan produk ke keranjang, serta melakukan proses checkout. Project ini dibuat untuk memenuhi Tugas Besar mata kuliah Technopreneurship.

Tujuan Project

1. Mensimulasikan konsep bisnis digital berbasis technopreneurship
2. Menyediakan tampilan website pemesanan kopi yang user-friendly
3. Mengimplementasikan fitur custom produk (kopi)
4. Membedakan hak akses antara customer dan admin
5. Menggabungkan konsep teknologi dan bisnis dalam satu prototype

Fitur Utama

* Melihat menu kopi

* Menambahkan kopi ke keranjang
* Custom kopi (level kopi, level gula, suhu, catatan khusus)

* Dashboard Admin (mockup)
* Manajemen stok (mockup)
* Admin tidak memiliki akses ke keranjang dan checkout
